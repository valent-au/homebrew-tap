# Valent-au Tap

## How do I install these formulae?

`brew install valent-au/tap/<formula>`

Or `brew tap valent-au/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "valent-au/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
